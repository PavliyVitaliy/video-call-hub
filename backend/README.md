## Description

Backend for video-call-hub

## Project setup

Create `.env` file based on `.dev.env` file with your settings

```bash
$ yarn install
```

Or you can use `Dockerfile` to launch the container

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
